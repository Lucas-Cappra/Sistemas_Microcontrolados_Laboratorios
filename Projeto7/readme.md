# **Projeto 7 - Modulador**

O objetivo deste projeto é desenvolver um radio definido por software, para realizar 4 tipos de modulações e uma interface IHM com um display LCD mostrando a configuração atual.


## Simulação no Simul IDE
A simulação no simulIDE pode ser realizada carregando o arquivo .sim no software, e carregando o código main.c no firmware diretamente, como alternativa também é possível utilizar o microchip studio para construir o arquivo .hex que pode ser carregado facilmente no firmware de maneira similar.


## Simulação em Hardware
Para a simulação em Hardware, é necessário um arduíno e um circuito pré-montado de acordo com os requisitos do projeto.
